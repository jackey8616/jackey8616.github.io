### Docky  
這個真的是一個很棒的Desktop工具, 其中幾個設定調整好之後，根本macOS。  
```sh
$ sudo add-apt repository ppa:docky-core/stable && apt-get update
$ sudo apt-get purge docky
$ sudo apt-get install docky    
```
