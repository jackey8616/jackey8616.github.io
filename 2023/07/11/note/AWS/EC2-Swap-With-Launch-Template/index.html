Use EBS as Swap at EC2

Posted on 2023-07-11 Edited on 2025-10-04 In AWS Views:

在使用Amazon Linux AMI中的EC2使用EBS作為Swap。 並結合Launch Template。

Opening

Amazon提供的EC2 optimize Linux預設是沒有開啟Swap的， 所以需要手動開啟。

Manual

以手動開啟為例，首先EC2要具備連入能力。 這邊不多贅述，可以自行搜尋EC2以SSH連入或是SessionManager等。

Check Swap

檢查Swap是否開啟的指令。

List block devices

列出目前所有的Disk以及Partition。

上圖為兩個EBS Volume分別為30G以及10G。 30G做為系統碟掛載在root，10G做為Swap磁區。

Hint: nvme1n1
